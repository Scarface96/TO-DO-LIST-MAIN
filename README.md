# TO-DO LIST

A feature-rich task management application for organizing and tracking daily tasks and projects.

## 📋 Overview

A user-friendly to-do list application built with modern web technologies to help you stay organized and productive.

## ✅ Features

- **Add Tasks** – Create new to-do items
- **Edit Tasks** – Update task descriptions
- **Delete Tasks** – Remove completed or unwanted tasks
- **Mark Complete** – Check off finished items
- **Categories** – Organize tasks by category
- **Priority Levels** – Set task importance
- **Due Dates** – Track deadlines
- **Local Storage** – Persist tasks in browser
- **Search/Filter** – Find specific tasks quickly
- **Responsive Design** – Works on all devices

## 🛠️ Technologies Used

- **HTML5** – Markup
- **CSS3** – Styling and layout
- **JavaScript (Vanilla)** – Task logic
- **LocalStorage** – Data persistence
- **No Dependencies** – Pure vanilla code

## 🚀 Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/Scarface96/TO-DO-LIST-MAIN.git
cd TO-DO-LIST-MAIN

# Open in browser
open index.html  # macOS
# or
start index.html  # Windows
# or
xdg-open index.html  # Linux
```

Or run a local server:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

Then visit `http://localhost:8000`

## 📖 How to Use

1. **Add Task** – Type task and click add
2. **Set Priority** – Choose importance level
3. **Add Due Date** – Set deadline
4. **Categorize** – Assign to category
5. **Mark Complete** – Check box when done
6. **Edit** – Click to modify task
7. **Delete** – Remove task from list
8. **Filter** – View tasks by category or status

## 🎨 Customization

Edit `style.css` to:
- Change colors and themes
- Modify layout
- Adjust fonts
- Add animations

## 💾 Data Storage

Tasks are saved to your browser's LocalStorage, so they persist even after closing the browser.
